# TEMPLATE USAGE GUIDE / テンプレート利用ガイド

## Purpose / 目的
- EN: Provide a repeatable way to apply these templates in any project.
- JA: どのプロジェクトでも再現可能な適用手順を提供する。

## Recommended Order / 推奨順序
1. `PROJECT_GOVERNANCE_AND_RULE_TEMPLATE.md`
2. `PROJECT_GOVERNANCE_GUIDELINE_FORMAL_TEMPLATE.md`
3. `DOCUMENT_CATALOG_TEMPLATE.md`
4. `PROJECT_STATUS_TEMPLATE.md`
5. `WIP_TEMPLATE.md`
6. `ISSUE_LIST_TEMPLATE.md`
7. `TEST_PLAN_TEMPLATE.md`

## Application Steps / 適用手順
1. EN: Copy the selected templates into the target project `docs/`.
   JA: 対象プロジェクトの `docs/` にテンプレートをコピーする。
2. EN: Replace placeholders with project-specific values.
   JA: プレースホルダをプロジェクト固有値に置換する。
3. EN: Update cross-links and ownership.
   JA: 相互参照と責任者を更新する。
4. EN: Verify UTF-8 and no mojibake when Japanese text exists.
   JA: 日本語を含む場合は UTF-8 と文字化けを確認する。

## Rule / 運用ルール
- EN: Templates are starting points only.
- JA: テンプレートは初期値であり、導入後の更新が前提です。
- EN: Each project must keep its own documents updated continuously.
- JA: 各プロジェクトで継続的に更新してください。
