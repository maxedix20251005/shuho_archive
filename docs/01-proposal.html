1 プロジェクト概要

サイトの目的（ビジネスゴール） 必須

1. 品川区内での認知獲得: 「品川区 書道教室」「書道 初心者」など地域×目的検索で比較候補に入る状態を作る。

2. 信頼形成の可視化: 塾の歩み、師範情報、作品実績を連動して提示し、初回訪問で不安を減らす。

3. 生徒募集の安定化: 主要CVを「問い合わせ送信」に統一し、月次の相談件数を安定確保する。

4. 運用負荷の最適化: 管理画面は「作品管理 / 問い合わせ管理 / お知らせ管理」の3機能に限定する。

ターゲットペルソナ（複数） 必須
ペルソナ属性目的重視点主CV
小学生の保護者30〜40代 / 品川区在住集中力・礼儀・文字改善通塾しやすさ、月謝、安全性問い合わせ
社会人初心者20〜40代 / 会社員趣味・リフレッシュ・美文字平日夜枠、初心者対応、持ち物問い合わせ
シニア層60〜70代生涯学習・脳活性昼間枠、落ち着いた環境電話または問い合わせ
検定志向層学生〜社会人段級位・資格取得指導実績、添削体制、上達計画問い合わせ

2 背景と課題

ユーザーニーズ・課題 必須

・ニーズ: 初心者でも通えるか、費用と時間が明確か、講師の質が分かるか、アクセスしやすいか。

・課題: 情報が分散すると比較が難しく、問い合わせ前に離脱が起きる。

・課題: 作品と講師情報が切り離されると、上達イメージが湧きにくい。

・課題: スマホ閲覧時に必要情報へ最短到達できないとCV率が下がる。

競合傾向（一般的） 必須
観点一般傾向不足しやすい点
訴求内容講師実績・受賞歴を強く訴求初心者向け導線が弱い
情報設計ページはあるが比較情報が分散料金・時間割・アクセスの統合不足
作品表示画像掲載中心カテゴリ整理・更新頻度が不十分
導線設計電話中心または単一フォーム不安解消後の自然なCV導線が弱い

3 提案するソリューション

差別化ポイント 必須

1. 「歩み × 師範 × 作品」を連続表示し、信頼形成を1スクロールで完結させる。

2. コースを対象別に整理し、比較表で月謝・時間・対象を同時提示する。

3. 作品カテゴリを「師範1/2/3・一般・こども」で統一し、閲覧意図に合う導線を確保する。

4. Heroは全面画像＋中央コピーで世界観を固定し、過剰な導線を排除する。

サイトコンセプト（世界観・方向性） 必須

コンセプト: 「一筆に、品格と心を。心を整える書の時間を、品川で。」

方向性: 参考サイト（maruoka-castle.jp）に学び、余白・写真・静けさを活かした上質な和のトーンで統一する。

体験価値: 初回訪問で“安心して始められる本格書道”を直感理解できる情報配置にする。

4 必要ページと導線

必要ページ構成（理由付き） 必須
ページ主目的理由
トップ印象形成・導線集約価値訴求と主要導線の起点
塾について信頼形成歩み・理念で継続運営の安心感を提示
師範一覧人的信頼講師の得意分野・指導方針を提示
コース・料金比較検討対象別の費用・時間を一画面比較
作品ポートフォリオ品質証明成果物で上達イメージを具体化
FAQ不安解消初心者不安・持ち物・欠席対応を先回り
アクセス来塾ハードル低減駅徒歩・地図・時間を明確化
お問い合わせCV獲得主要コンバージョン地点
コンバージョン設計（問い合わせ導線） 必須

・主要CV: 問い合わせフォーム送信（フォーム完了イベントを計測）。

・補助導線: 電話（シニア層向け）をヘッダーとアクセスページに常時表示。

・配置ルール: 主要ページごとに「ファーストビュー下 / 本文中段 / フッター手前」の3点で問い合わせ導線を設置。

KPI（6か月目標） 必須
KPI目標値計測方法
月間セッション1,500〜2,000GA4
問い合わせ件数月30件フォーム送信 + 電話計測
フォームCVR1.8%以上CV数 / セッション
主要ページ読了率40%以上スクロールイベント
問い合わせ→入塾率25%以上運用台帳突合
