2 ページ一覧

Webサイト ページ一覧 必須
No.ページ名ページの目的主要コンテンツ
1トップ第一印象形成と主要導線への誘導ヒーロー、塾の価値、注目作品、CTA、アクセス要約
2塾について信頼形成塾の歩み、理念、指導方針、代表メッセージ
3師範一覧講師理解師範プロフィール、得意分野、担当コース
4コース・料金比較検討支援対象別コース、月謝、時間割、持ち物、入会フロー
5作品ポートフォリオ品質証明作品画像、カテゴリ絞り込み、制作背景
6生徒募集要項募集条件の明確化募集対象、定員、受付期間、注意事項
7FAQ不安解消初心者可否、道具、欠席振替、服装、支払い方法
8アクセス来塾率向上地図、最寄駅、徒歩ルート、外観写真
9体験申込主要CV獲得申込フォーム、希望日時、同意チェック
10問い合わせ相談窓口問い合わせフォーム、電話番号、LINE導線
Webアプリ 画面一覧 必須
No.画面名画面の目的主要機能
1管理トップ運用状況の把握新着件数、未対応件数、ショートカット
2問い合わせ一覧問い合わせ対応検索、絞り込み、ステータス更新
3体験申込一覧申込対応管理希望日時確認、対応メモ、対応状態更新
4作品管理掲載作品の更新新規登録、編集、公開/非公開切替
5師範情報管理講師情報の保守プロフィール編集、表示順制御

3 ユーザーフロー

メインシナリオ 必須
ペルソナがサイトに訪問してから目的を達成するまでの動線（メインの行動パターン）

検索（「品川区 書道教室 初心者」）→ トップ閲覧 → 塾の歩み/師範一覧で信頼確認 → コース・料金で比較検討 → FAQで不安解消 → 体験申込フォーム送信 → 完了ページ到達

サブシナリオ 任意
その他の行動パターン（1〜2パターン）

サブ1：SNS流入 → 作品ポートフォリオ閲覧 → 師範一覧 → 問い合わせ送信

サブ2：Googleマップ流入 → アクセス確認 → 電話問い合わせ → 体験予約確定
