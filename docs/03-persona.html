1 基本プロフィール

ペルソナA（一次ターゲット） 必須

氏名: 田中 美咲（仮） / 34歳 / 女性 / 品川区在住

職業: 会社員（時短勤務）

家族構成: 配偶者・小学生の子ども1名

目的: 子どもの集中力と文字改善、礼儀作法を身につけたい

最終行動: 問い合わせフォーム送信

ペルソナB（一次ターゲット） 必須

氏名: 鈴木 恒一（仮） / 29歳 / 男性 / 港区勤務

職業: IT企業勤務

目的: 趣味とメンタルリセット、美文字習得

重視点: 平日夜の通塾可否、初心者歓迎、通いやすさ

最終行動: 問い合わせフォーム送信

ペルソナC（二次ターゲット） 必須

氏名: 佐々木 和子（仮） / 67歳 / 女性

目的: 生涯学習と仲間づくり、生活リズムの維持

重視点: 昼間帯、落ち着いた環境、電話相談のしやすさ

最終行動: 電話または問い合わせフォーム

2 ライフスタイル

平日行動パターン 必須

保護者層: 朝に情報収集、夜に比較検討し、週末に家族と相談して意思決定。

社会人層: 通勤時間と就寝前にスマホで比較、短時間で候補を絞る。

シニア層: 日中にPCまたはタブレット閲覧、電話相談への安心感が高い。

関心・価値観 必須

・「続けられること」を重視し、上達の見通しがある教室を選ぶ。

・価格の安さだけでなく、講師品質と安心感で判断する。

・世界観に共感できるサイトは、問い合わせ行動につながりやすい。

3 デジタル環境

利用デバイス・行動 必須

主デバイス: スマホ（全体の約7〜8割を想定）

補助デバイス: PC（詳細比較時）

主な流入: Google検索、Googleマップ、SNS投稿、口コミ共有

期待値: 3分以内に必要情報へ到達し、問い合わせ送信まで完了できること

4 ニーズと課題

ニーズ 必須

1. 初心者でも安心できる説明（難易度、持ち物、当日の流れ）。

2. 比較しやすい情報（料金、時間、対象、アクセス）。

3. 講師と作品から教室品質を確認できること。

4. 迷わず相談できる問い合わせ導線。

課題 必須

1. 必要情報が別ページに散ると離脱率が上がる。

2. 作品の分類が曖昧だと自分向けの成果イメージが持てない。

3. 問い合わせフォームの入力負荷が高いとCVが落ちる。

4. スマホで視認性が低いと信頼形成前に離脱する。
