1 デザインコンセプト

基本方針 必須

参照構成: 強いヒーロー + ストーリー主導の縦スクロール構成

ページリズム: ヒーロー → 価値訴求ストーリー → 見どころ → お知らせ → 実用情報 → フッター

ブランドキーワード: 静謐 / 品格 / 親しみ / 継承 / 成長

対象配慮: 初心者・子ども・社会人が入りやすい、威圧感のない表現に統一する

文言運用: 抽象表現のみで終わらせず、歩み・実績・作品・FAQなど根拠情報を併記する

4 UIコンポーネント

ヒーロー 必須

全面画像 + 中央キャッチコピー（見出し1行、補助1行）を基本とする。

現スコープではヒーロー内に体験申込CTAを置かない。

カード（師範 / 作品） 必須

カード余白、角丸、メタ情報階層を共通化する。

師範ページは「師範3名・1段構成」を標準とする。

フォーム / CTA 必須

必須マーク、入力エラー、返信目安（1〜2営業日以内）を明示する。

問い合わせフォームを主要コンバージョン地点として設計する。

5 レイアウトルール

余白・階層・アクセシビリティ 必須

セクション余白: PC 80-120px / SP 48-72px

ナビ順序固定: 塾について / 師範 / コース / 作品 / FAQ / アクセス / お問い合わせ

コントラスト: 本文4.5:1以上を目標とする

画像代替文: 意味のある画像には日本語altを必須化する

運用連携: スコープ変更時は `07-specification` と同時更新する
